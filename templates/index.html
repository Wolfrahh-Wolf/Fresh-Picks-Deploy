Algorithmic Core

Dispatch & Routing

Min-Heap Priority Queue

Priority-Based Sorting

Powers the admin order dispatch engine. Queued orders are sorted by delivery slot preference - Morning slots bubble to the heap root for immediate extraction and assignment.

Round-Robin Assignment

Round-Robin

Cycles through available delivery agents one by one, ensuring every agent gets an equal share of orders. No one gets overloaded, no one sits idle.

Engineering Excellence

Security Layer

Password Hashing

SHA-256 Cryptography

Plaintext credentials are cryptographically hashed in Python before being written to the database, ensuring stored passwords are never exposed as readable strings.

OTP Verification

Time-To-Live Logic

Secure, time-limited One-Time Passwords guard critical operations — account registration, password changes, and order cancellations — with email-based 2-step verification.

Automated Mailing & Integrations

Receipt Dispatch

SendGrid SMTP

Instantly packages the generated PDF invoice and emails it directly to the customer's inbox upon a successful checkout — fully automated, zero manual steps.

OTP Dispatch

SendGrid API

OTP codes for registration, password changes, and cancellations are instantly composed and relayed to the customer's inbox via the SendGrid transactional API.

Payment Gateway

Razorpay · HMAC-SHA256

A secure 2-step checkout pipeline using server-side HMAC-SHA256 signature validation to prevent client-side cart spoofing before any order is committed.

Dynamic Receipts

FPDF · Python

High-fidelity PDF invoices generated on the fly, embedding brand assets, exact timestamps, itemised totals, and delivery slot details for every completed order.
