Algorithmic Core (DSA)

Data Structures

Singly Linked List

O(N) Time | O(N) Space

Serves as the foundational memory architecture. Dynamically deserializes database records directly from flat binary .dat files at runtime.

Doubly Linked List

O(1) Mid-List Deletions

Drives the real-time shopping cart. Bi-directional node traversal enables instant item removals without array-shifting overhead.

Min-Heap Queue

O(log N) Time

Powers the Admin Orders & Dispatch Engine. Sorts queued orders by delivery urgency (e.g., Morning slots bubble to the root) for immediate extraction.

Circular Linked List

Infinite Loop Routing

Facilitates the logistics engine. Implements a wrap-around round-robin algorithm to guarantee mathematically equitable order assignment.

Search Algorithms

Hash Table

O(1) Lookup / Perfect Hashing

Custom lookup table. Parses alphanumeric IDs (e.g., U1042 → 42) into array indices to instantly map to specific memory addresses, completely bypassing slow O(N) SLL linear searches.

Unique Engineering Features

Security & Networking

HTTPS Security

Self-signed X.509 SSL

Ensures all local traffic is encrypted via TLS, allowing secure interactions with external APIs and preventing packet sniffing.

Password Hashing

SHA-256 Cryptography

Plaintext credentials are cryptographically hashed in Python before reaching the C-engine, ensuring binary databases remain secure.

Intranet Hosting

0.0.0.0 Network Binding

Dynamically detects the host's local IPv4 address to broadcast the Flask server for seamless cross-device testing on the same network.

OTP Verification

Time-To-Live Logic

Generates secure, time-limited One Time Passwords (OTPs) for safe user registration and account recovery.

Automated Mailing & Integrations

Receipt Dispatch

C-Based SMTP Engine

Instantly packages the generated PDF invoice and emails it directly to the customer's inbox upon a successful checkout.

OTP Dispatch

Direct SMTP Relay

Instantly wraps the OTP and emails it directly to the customer's inbox on crucial operations.

Payment Gateway

Razorpay API

A secure 2-step checkout pipeline that utilizes server-side HMAC-SHA256 signature validation to prevent client-side cart spoofing.

Dynamic Receipts

FPDF Python Library

Generates high-fidelity PDF invoices on the fly, embedding custom brand icons, exact timestamps, and specific transaction details.
